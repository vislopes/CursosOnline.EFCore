# CursosOnline.EFCore

Sistema de gestão de cursos online desenvolvido em C# utilizando Entity Framework Core.

## Funcionalidades
- Cadastro de cursos
- Cadastro de alunos com validação de email único
- Matrícula de alunos em cursos
- Listagens com LINQ
- Persistência de dados com EF Core
- Versionamento de banco com Migrations

## Tecnologias
- C#
- .NET
- Entity Framework Core
- SQLite
- LINQ
